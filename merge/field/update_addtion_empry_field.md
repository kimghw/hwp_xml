# case1 
1. 필드이름이 없고, 빈 셀의 바로 위 행이 모두 data_, input_ 인 경우
2. 그리고 컬럼의 수와 너비가 동일한 경우
3. 모두, 위 셀의  nc.name 과 동일하게 삽입한다. 
# case2
1. 필드이름이 없고, 1개 이상의 위아래가 병합된 경우
2. gstub 으로 하고 gstub_{랜덤} 으로 삽입한다.
3. 우측에 빈 셀이면 첫번째 행의 셀은 input_{랜덤} 으로 입력한다.
4. gstub_{랜덤}이 동일한 행은 두번재 부터  첫번째 행의 nc_name 값을 복사하여 사용한다.
# case3
1. gstub가 연속해서 2개열 이상에 겹쳐진 경우,
2. 빈셀의 nc_name 값은 case2와 동일하게 한다.
3. gstub1, gstub2, gstub3, 빈셀, 빈셀, 빈셀 인 경우 3개가 모두 동일한 경우에만 동일한 nc_name을 갖는다.

# gstub 은 동일한 nc.name을 갖지 않음
# 2개 이상의 gstub 를 갖는 행은 2개 gstub가 동일할 때만 같은 우측의 빈셀의 nc.name을 동일하게 갖을 수 있음
